整体开场（30 秒，任意一人）

我们做的是对标 lg.team 的 Scrum 敏捷管理工具，采用 React + Spring Boot 前后端分离架构，目前已完成 12 个功能模块，联调脚本 30/30 全部通过。

技术栈：

  • 前端：React + TypeScript + Vite + Ant Design + Zustand + TanStack Query + @dnd-kit
  • 后端：Spring Boot 3 + MyBatis Plus + H2 + JWT + WebSocket

成员分工详情

负责模块一（全局）、模块二（认证与个人设置）

汇报重点

  1. 项目结构：fronted/、backend/、docs/、scripts/ 分工说明

现场汇报建议

  1. 每人带电脑，提前启动前后端（npm run dev + mvn spring-boot:run），避免现场装依赖。
  2. 按 A → B → C → D 顺序演示，形成「登录 → 工作台 → 看板 → 脑图」完整故事线。
  3. 每人准备 2～3 个代码文件即可，优先展示自己负责模块的 Service + Page。
  4. 统一口径：目前为中期版本，核心 Scrum 链路已通，脑图已重构；后续可做性能优化、移动端适配、权限细粒度等。
