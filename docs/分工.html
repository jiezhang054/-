任务书概要

设计并实现一个轻量级、聚焦 Scrum 核心流程、支持实时协作的敏捷项目管理工具，对标 lg.team，覆盖产品待办列表、冲刺管理、任务看板、燃尽图，并实现产品路线图 → 里程碑 → Sprint 看板自动生成链路。

核心需求边界（任务书）

  • Scrum 核心：看板、泳道、卡片拖拽、燃尽图、权限 RBAC
  • 自动生成：里程碑看板、Sprint 看板由路线图规划自动创建
  • 实时协作：WebSocket 推送，多人编辑数据一致性
  • 工作台：近期事项、星标看板、最近访问、卡片动态
  • 脑图：项目共享脑图，节点编辑与导入导出
说明：任务书原计划前端为 Vue3 + Element Plus，实际实现采用 React + TypeScript + Vite + Ant Design，后端为 Spring Boot 3 + JWT + H2 + WebSocket，功能范围与任务书一致。

当前进度：已完成 12 个功能模块开发，联调脚本 30/30 通过，处于 Sprint 2 末～Sprint 3 初（中期检查节点）。
